# Smart-Classroom-Timetable-Scheduler
Smart Classroom and Timetable Scheduler is a web-based system that automates classroom allocation and timetable generation. It efficiently schedules classes, faculty, and resources, reduces conflicts, optimizes room usage, and provides real-time timetable updates for students, teachers, and administrators.
Smart Classroom & Timetable Scheduler

AI-powered Smart Classroom and Timetable Scheduler built using React, FastAPI, MongoDB, and Machine Learning to automate classroom allocation and timetable generation.
